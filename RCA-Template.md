# Root Cause Analysis (RCA) Template

**Incident Reference:** INC-XXXXX  
**Date:** YYYY-MM-DD  
**Severity:** High/Critical  

### 1. Problem Description
Clear description of the service interruption affecting enterprise systems.

### 2. Root Cause
Technical discovery of why the failure occurred (e.g., configuration drift, expired certificate, failed patch).

### 3. Resolution Steps
Step-by-step actions taken to restore service within SLA targets.

### 4. Preventative Actions
New Knowledge Base (KB) entries or monitoring alerts implemented to prevent recurrence.
